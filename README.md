# Atividades e trabalhos — PHP
Esse repositório contém todas as atividades, trabalhos e projetos que realizei em PHP nos anos de 2024 e 2025. Contém desde as aulas iniciais com exercícios simples até os trabalhos maiores integrados ao banco de dados. Nessas pastas contém arquivos de HTML, CSS, PHP, uso do banco de dados MySQL e alguns trabalhos também foi pedido PHP para conectar na internet na disciplina de Arduino, onde além da parte de Arduino eu tinha que conectar no Wi-Fi e realizar um site em PHP que conectasse com ele.
